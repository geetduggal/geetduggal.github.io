Highlights synced from Readwise — books, articles, podcasts. Entries keep their original frontmatter (title, author, image) so the card grid renders covers straight from Amazon image URLs.

    - or:
        - type.contains("article")
        - type.contains("tweet")
        - type.contains("podcast")
        - type.contains("book")
views:

    name: Recent
    sort:
      - property: published
        direction: DESC
